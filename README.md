## CSS Wiki

Небольшой справочник по базовым CSS-свойствам для layout'а в формате двухколоночного сайта.

### Структура секций правой колонки

В правой колонке расположены секции с такими `id`:

- `display`
- `position`
- `flex`
- `grid`
- `margin`

Ссылки в левой навигационной колонке (`<nav>`) указывают именно на эти `id`, поэтому при клике происходит скролл к соответствующей карточке свойства.
