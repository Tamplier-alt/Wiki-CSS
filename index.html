Мини-справочник по основным CSS-свойствам для построения макета.

Быстрые переходы: display position flex grid margin

display

Свойство display определяет тип форматирования элемента и то, как он участвует в потоке документа.

Примеры значений: block, inline, inline-block, flex, grid.

position

Свойство position задаёт способ позиционирования элемента и от чего будут считаться координаты.

Примеры значений: static, relative, absolute, fixed, sticky.

flex

Свойство flex — это сокращение для flex-grow, flex-shrink и flex-basis и управляет размером flex-элементов внутри flex-контейнера.

Примеры значений: 1, 0 1 auto, 2 0 200px.

grid

Свойство grid — шорткат для настройки рядов и колонок CSS Grid-контейнера, объединяющий несколько grid-свойств.

Примеры значений: auto / auto, 100px 100px / 1fr 2fr, комбинации с repeat().

margin

Свойство margin задаёт внешние отступы вокруг элемента и влияет на расстояние между блоками.

Примеры значений: 0, 16px, 10px 20px, auto.
